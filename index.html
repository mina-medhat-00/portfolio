Notable Projects

Softline

Softline is an all-in-one business automation platform designed to revolutionize service-based operations using React Native, Nest.js, and Prisma. I spearheaded the integration of AI-powered virtual receptionists and sales coaching via Retell AI, creating a system that autonomously captures, routes, and manages lead calls to maximize conversion rates. Beyond communication, the platform streamlines field logistics through sophisticated scheduling and dispatching tools for subcontractors, paired with real-time inventory management. By centralizing client interactions within a unified CRM, the solution enables seamless appointment management and automated workflows via Make.com, providing a robust end-to-end infrastructure for modern service businesses.

Workflower

Workflower is an intelligent LinkedIn outreach engine built with TypeScript that automates the entire lead generation lifecycle. By leveraging the Stagehand SDK and Playwright for advanced browser automation, the system handles complex session management and secure browsing via GoLogin to scale networking safely. I integrated the OpenAI API to develop a high-precision filtering system that autonomously qualifies prospects based on professional data and generates hyper-personalized messaging. This project showcases the powerful intersection of web automation and LLMs, transforming manual prospecting into a data-driven, autonomous growth machine.

Mock-Mate

Mock-Mate is a comprehensive technical interview platform developed in a collaborative team environment using React, Laravel, and Inertia.js. The application facilitates real-time assessments through WebRTC video integration via LiveKit and a shared, collaborative coding environment with instant execution capabilities. I was responsible for architecting the robust role-based access control (RBAC) system and the complex session/queue management logic that ensures a smooth user flow. Additionally, I built an extensive analytics dashboard to provide users with deep insights into interview performance metrics and activity, bridging the gap between practice and professional readiness.

VaniFit

VaniFit is a high-performance e-commerce application that delivers a fluid, near-SPA user experience using Vanilla JavaScript and Bootstrap. Utilizing a serverless architecture with Firebase and Firestore, the platform features real-time data synchronization and secure user authentication. I implemented robust backend validation and a full-featured product management system, including dynamic inventory tracking and order history. By integrating Stripe for secure payment processing, VaniFit demonstrates the ability to build scalable, secure, and highly responsive web applications with a focus on clean code and optimized frontend performance.
